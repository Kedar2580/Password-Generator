# 🔐 Random Password Generator

A simple and powerful **Random Password Generator** built using **HTML, CSS, and JavaScript**. This web-based tool helps users create strong, secure, and customizable passwords instantly.

---

## 📌 Features

* Generate strong and random passwords
* Customize password length
* Include/exclude:

  * Uppercase letters (A–Z)
  * Lowercase letters (a–z)
  * Numbers (0–9)
  * Special characters (!@#$%^&*)
* One-click **Copy to Clipboard**
* Password Strength Indicator
* Responsive and user-friendly UI

## 💡 How It Works

The password is generated using JavaScript's randomization logic. Based on user-selected criteria, the program:

1. Creates a pool of valid characters
2. Randomly selects characters using `Math.random()`
3. Combines them to form a secure password

---

## 🔒 Password Strength Logic

The strength indicator is based on:

* Password length
* Variety of character types used

Stronger passwords include a mix of uppercase, lowercase, numbers, and symbols.
